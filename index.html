HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Section elements divide the main content into subjects.
  • The h2 element represents the heading different from h1 to differentiate the sections.
  • The li element creates bullet lists like these!

CSS

  • margin: indicates how much space we want around the outside of an element
  • padding: indicates how much space we want around the content inside an element
  • width, height: specify the size of elements in either % or px
  • text-align: aligns the text within the element
  • color, background-color: specify the color of the element or its background, respectively
  • font-size: sets the size of the text font

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git status: shows the current branch and if there are new changes
  • git commit: moves changes into the repository
  • git push: updates remote repository
